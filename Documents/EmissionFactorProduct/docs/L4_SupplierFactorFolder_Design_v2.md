# L4 供應商係數資料夾 - 畫面及功能規劃 v2

## 一、系統架構說明

### 1.1 資料流程
```
L4 產品模組（資料收集專案）
    ↓ [專案完成並鎖定]
    ↓ [勾選匯入L3]
    ↓ [自動同步]
L4 供應商係數資料夾 ← [新資料通知🔴]
    ↓
L3 中央係數庫
```

## 三、核心功能規劃

### 3.1 資料同步機制
- **自動同步**：L4產品模組專案鎖定後自動推送
- **同步通知**：紅點提示 + 數量顯示
- **同步追蹤**：記錄來源專案、時間、負責人

### 3.2 審核工作流
- **批次審核**：支援多選快速審核
- **逐一審核**：詳細查看每個產品係數
- **審核狀態**：新同步🔴、審核中🟡、已通過✅、已退回❌

### 3.3 L3匯入管理
- **批次匯入**：選擇多個已審核係數
- **智慧分類**：自動建議ISIC分類
- **匯入追蹤**：記錄匯入時間、操作人員

### 3.4 資料檢視功能
- **多維度篩選**：區域、年度、同步狀態、L3狀態
- **歷史追溯**：查看資料來源專案
- **版本比較**：同產品不同年度對比

### 3.5 狀態管理
```
新同步(🔴) → 審核中(🟡) → 已審核(✅) → 已匯入L3(✓)
                    ↓
                已退回(❌)
```

## 五、互動流程優化

### 5.1 新資料提醒流程
1. L4產品模組專案完成並鎖定
2. 系統自動偵測並同步資料
3. 供應商係數資料夾顯示紅點通知
4. 用戶點擊查看新同步資料
5. 進入批次審核或逐一審核流程

### 5.2 快速處理流程
1. 查看新同步通知面板
2. 批次選擇信任的供應商
3. 一鍵批次審核通過
4. 直接進入L3匯入流程
5. 完成匯入並更新狀態

## 六、使用者體驗優化

### 6.1 視覺提示
- **紅點通知**：新資料同步提醒
- **顏色編碼**：不同狀態使用不同顏色
- **進度指示**：顯示處理進度

### 6.2 操作優化
- **批次操作**：減少重複動作
- **快捷鍵**：支援鍵盤快速操作
- **記憶功能**：記住用戶的篩選偏好

### 6.3 資訊透明
- **完整追溯**：從原始專案到L3的完整路徑
- **歷史記錄**：所有操作都有記錄可查
- **狀態即時**：即時更新各種狀態變化
